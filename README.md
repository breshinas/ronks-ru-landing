# ronks.ru — лендинг агрегатора автозапчастей

Разработка веб-сайта лендинга агрегатора автозапчастей **ronks.ru** с поиском аналогов запчастей (кроссов).

## Краткое описание

Цель проекта — предоставить пользователям простой и быстрый способ поиска и подбора аналогов автозапчастей (кроссов) по номеру детали, производителю или автомобилю. Лендинг будет содержать:
- краткое описание сервиса ronks.ru;
- форму поиска запчастей и их аналогов;
- блоки преимуществ и доверия (бренды, отзывы, гарантии);
- контактную информацию и ссылки на дополнительные разделы.
